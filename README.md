# NeubotTests

Collection of download/upload HTTP speedtests obtenined through the [Neubot project](http://www.neubot.org).

### Installation 

```sh     
unzip "speedtests/*.zip" -d speedtests/
sudo rm -r .git speedtests/*.zip speedtests/__MACOSX  
```
